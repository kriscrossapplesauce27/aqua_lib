A library for Kristal that adds Aqua from DELTARUNE Chapter 5 as a party member.

To use the colored text and miniface use this [color:#65f8ff]  text command and this one [miniface:aqua], 
you can also choose the battle head icon to be the miniface or a custom one in lib.json .
